# Typetchi v0.6.0

Version: 0.6.0

## Release focus

- Synchronize package, extension manifest, and built manifest versions to `0.6.0`.
- Treat the header close action as a recoverable collapse action until a dedicated extension popup restore entry exists.
- Normalize legacy widget storage so old `closed: true` data cannot make Typetchi permanently disappear.

## Verification

Run these checks before loading the unpacked extension:

```sh
npm run typecheck
npm run build
npm run check:version
```

Then reload the unpacked `dist/` directory from `chrome://extensions` and confirm Chrome shows version `0.6.0`.
