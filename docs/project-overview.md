# Typetchi Project Overview

## 1. Typetchi 是什麼

Typetchi 是一個 Chrome Extension，提供一個常駐在網頁上的療癒小視窗。

使用者在一般網頁中手動打字時，Typetchi 會將有效輸入轉換為角色經驗值。角色會依照經驗值、打字行為與未來條件逐步成長，形成類似電子寵物的陪伴體驗。

Typetchi 的核心不是單純打字速度工具，而是「靠日常打字慢慢養成的小夥伴」。

---

## 2. 核心產品目標

Typetchi 的目標是：

1. 讓使用者在日常瀏覽與工作中，感受到輕量、療癒的陪伴。
2. 將打字行為轉換成可視化成長，例如 EXP、Level、Stage、進化。
3. 鼓勵使用者穩定輸入，而不是靠貼上文字快速刷進度。
4. 讓角色成長系統具有長期擴充性，例如分支進化、任務、成就、多角色。
5. 保持對宿主網站的低干擾，不影響原本頁面功能。

---

## 3. 目前主要功能

Typetchi 目前包含：

- Chrome Extension Manifest V3 架構
- Content Script 注入療癒小視窗
- Shadow DOM 隔離樣式
- Widget 拖拉、縮放、固定、收合、關閉
- 使用 `chrome.storage.local` 保存狀態
- 跨分頁狀態同步
- EXP / Level / Stage 成長系統
- 三階段進化
- 角色動畫與互動回饋
- 今日輸入統計
- CPM / WPM 打字速度統計
- 貼上文字不計入 EXP
- 重置角色進度
- Extension popup 恢復關閉後的 Widget
- 版本號同步檢查

---

## 4. 產品原則

Typetchi 必須遵守以下原則：

1. 只統計字數與狀態，不保存使用者實際輸入內容。
2. 貼上的文字不計入 EXP、任務、速度或成長進度。
3. 不讀取 clipboard 文字內容。
4. 不統計 password、email、tel、number 等敏感欄位。
5. 不干擾宿主網站原本的輸入、貼上、快捷鍵、滾動、按鈕與連結。
6. UI 必須存在於 Shadow DOM 中，避免污染宿主頁面。
7. 所有重要狀態需使用 `chrome.storage.local` 保存。
8. 已開啟的多個分頁需盡量保持 Typetchi 狀態同步。
9. 新功能需避免造成 Widget 跑版或遮擋頁面主要操作。
10. 若功能會完全隱藏 Widget，必須提供明確的恢復入口。

---

## 5. 目前不做什麼

Typetchi 目前不做：

- 不做後端服務
- 不做登入系統
- 不做雲端同步
- 不做排行榜
- 不做完整打字練習平台
- 不做 AI 對話寵物
- 不保存使用者輸入內容
- 不分析使用者輸入文字語意
- 不讀取使用者剪貼簿內容
- 不主動修改宿主網站內容
- 不攔截宿主網站原本事件

---

## 6. 未來方向

Typetchi 未來可以逐步擴充：

1. 分支進化系統  
   根據使用者的打字習慣、速度、連續天數、任務完成度，進化成不同型態。

2. 每日任務系統  
   例如今日手打 300 字、維持 30 CPM、連續輸入 5 分鐘。

3. 成就系統  
   例如第一次進化、累積 10000 字、連續 7 天使用、達到特定 CPM。

4. 多角色與角色圖鑑  
   讓使用者解鎖、切換、收藏不同角色。

5. 設定頁與 Popup 強化  
   提供更完整的顯示、隱私、動畫、統計設定。

6. 上架準備  
   包含 README、隱私說明、權限檢查、Debug log 開關、Chrome Web Store 所需素材。
